Side Project

Browser Opener

Browser Opener is a macOS application to help you open browsers based on URL.

As a front-end engineer, I use Chrome for debugging applications but use Safari for other kinds of stuff. It's painful to switch between browsers when opening different URLs, for example, localhost or 127.0.0.1.

Browser Opener can help users decide which browser to open when clicking links. It now even supports browser profiles.

App Store: https://apps.apple.com/app/browseropener/id1387480496

Product Hunt: https://www.producthunt.com/posts/browseropener

Reading List

Reading List is an application for collecting web pages for later reading, which supports iOS, iPadOS, and macOS.

After trying several RIL services, I found they all contain features I don't need. Also, none of them supports iPad very well. So I created my own.

App Store: https://apps.apple.com/app/reading-list-read-later/id1501647738

Bangumi List

Bangumi List is a Japanese anime information site.

Back in China, due to the fragmentation of copyrights, it was hard to check which anime is on which video site. So I created this site to help Chinese anime fans to find out where to watch their favorite anime series.

This site is also my playground of front-end technologies. It has gone through a long journey: plain HTML, jQuery, Angular, React. Recently it's refactored to TypeScript and added the login system.

Bangumi List https://bgmlist.com

GitHub(v3): https://github.com/wxt2005/bangumi-list-v3
